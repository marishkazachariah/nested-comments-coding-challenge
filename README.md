### How to run the app

| Command             | Description                                               |
|---------------------|-----------------------------------------------------------|
| `npm run start`     | Runs in dev mode                                          |
| `npm run test`      | Runs Unit tests                                           |
| `npm run typecheck` | Runs Typescript checks                                    |
| `npm run lint:fix`  | Runs Eslint to automatically fix prettier & eslint errors |
| `npm run lint`      | Runs Eslint                                               |
